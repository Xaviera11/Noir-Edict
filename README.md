# Noir-Edict
A mystical web app that reveals the meaning, origin, and spiritual path behind your name.

## ✨ Features

- Name origin and cultural meaning lookup
- Spiritual interpretation based on name energy
- Optional Life Path Number from birth date
- AI‑powered fallback for names not in the database
- Beautiful, mystical dark theme with leopard spirit imagery

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Gemini API (via Google Apps Script proxy)
- Google Apps Script (secure backend)
